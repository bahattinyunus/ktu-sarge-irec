# Variable Nomenclature

Standard variable names used in the project codebase and documentation.

| Variable Description | Symbol / Variable Name |
| :--- | :--- |
| Yükseklik (Altitude) | `h` |
| Delta (Difference) | `d_` |
| Basınç (Pressure) | `pa` |
| Sıcaklık (Temperature) | `T` |
| Zaman (Time) | `t` |
| İvme (Acceleration) | `a` |
| Dikey İvme (Vertical Acc) | `a_z` |
| Yanal İvme (Lateral Acc Y) | `a_y` |
| Yanal İvme (Lateral Acc X) | `a_x` |
| Maksimum Yükseklik | `h_max` |
| Delta Yükseklik | `h_d` |
| Dikey Hız (Vertical Vel) | `v_z` |
